Feature overview

Everything on board

Formats & engine

  • Four formats — Americano, Mexicano, Americano Team, Mexicano Team
  • Smart pairing engine — minimizes repeated partners & opponents, perfect rotation when one exists
  • Mexicano re-seeding — 1st & 2nd vs 3rd & 4th, every round, from the live standings
  • Standings-seeded final round for every format
  • Fair rests — byes rotate automatically for any player count
  • Configurable match points — 16 / 21 / 24 / 32 / custom

Organizing

  • Five-step wizard with live validation and warnings
  • Player import — Playtomic exports, CSV or plain text
  • Roster quick-pick + team builder with shuffle
  • Courts your way — numbered or custom-named
  • Multi-club support with explicit club selection

Live play

  • A page per court — tablet-friendly score entry at the net
  • Fullscreen presenter view — all courts, round controls, ranking popup
  • Validated public scoring — sums always match the point total
  • Live standings with auto-refresh everywhere
  • Manager corrections for any round, any time

Results & rankings

  • Results matrix — every player × every round, for every tournament
  • Club ranking — 100/90/…/10 pts + 1 for participation
  • 365-day rolling window — always the last year of padel
  • Editable entries and manual point adjustments

Clubs & roles

  • Public club pages — tournaments, results, ranking
  • Club manager panel — roster, ranking, tournament control
  • Super admin panel — clubs, manager assignment with user search, admin accounts
  • Full audit log — every action recorded and filterable
  • Secure sign-in with Clerk

Experience

  • English & Polish — one click, auth screens included
  • Modern dark UI — navy & volt, consistent everywhere
  • Responsive — phone, tablet, laptop, club TV
  • Demo club included — 18 players, played tournaments, live ranking

Under the hood

A modern, production-grade stack

Next.js 15App Router with React Server Components — every page rendered fresh on the server, APIs and UI in one codebase.
React 19 + TypeScriptStrict TypeScript end to end, from the tournament engine to every screen.
Tailwind CSS v4A custom navy-and-volt design system: one look across public pages, panels and the presenter view.
ClerkSecure authentication with role-based access — super admins, club managers, players — localized in English and Polish.
MongoDB Atlas + MongooseCloud database storing clubs, rosters, tournaments with full round history, rankings and the audit log.
VercelDeployed globally on Vercel’s edge network — the app you’re reading about is live right now.
Built to be trusted The pairing engine is validated with automated simulations (perfect partner rotation, fair rests, correct Mexicano seeding) · UI flows verified with Playwright browser automation — the screenshots in this document were captured by it · dependencies patched against the latest security advisories (incl. CVE-2025-66478) · every mutation is authorization-checked and audit-logged.
